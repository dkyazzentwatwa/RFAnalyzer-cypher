CHANGELOG - RF Analyzer

version 2.3-dev
- Build and tooling:
  - Restore local FOSS debug build path with documented Android Studio JBR, SDK 36, NDK and CMake setup
  - Align native DSP CMake version with the rest of the native modules
  - Add reproducible MkDocs dependency list for generating the in-app manual
- Android compatibility:
  - Declare connected-device foreground service usage for native USB SDR sources
  - Allow large recording exports to continue when notification permission is denied
  - Persist file-source read grants when Android's file picker provides them
  - Start edge-to-edge handling for modern Android devices
- RF/SDR stability:
  - Fix HackRF LNA startup gain to use the configured LNA gain steps
  - Drop failed FFT conversion packets instead of forwarding stale data
  - Move HydraSDR native version and gain calls to current libhydrasdr APIs
- UI:
  - Refresh shared Material theme typography and component shapes

version 2.2
- Bookmarks:
  - Bookmarks for Stations and Bands
  - Bookmarks show as labels in FFT plot
  - Import from SDR#, SDR++ and legacy RF Analyzer 1.13 lists
  - Import (and auto-update) online station lists:
    - EiBi, POTA & SOTA Spots
  - Included Band Plans: Amateur Radio, ISM, Maritime and Air Bands
- Support for Airspy HF+ SDR
- RTL-SDR Direct Sampling Support (Experimental)
- Switched from hackrf_android to libhackrf v0.10.0
- Upgraded to libhydrasdr 1.1.1
- Low Performance Mode Switch: Reduce Filter Quality to run smoother on older devices
- Bug Fixes:
  - Export Recordings in Background Task to prevent Freeze (large recordings)

version 2.1
- Support for Airspy SDR
- Support for HydraSDR RFOne SDR
- Added Rational Resampler: Demodulation of arbitrary sample rates is now possible
- Switched audio output sample rate from 31250 Hz to 48000 Hz
- Automatically disable any Frequency restrictions when RTL-SDR Blog v4 is connected
- Multiple Bug fixes for app crashes and freezes

version 2.0
- Complete rewrite of the app from scratch
- New modern Material Design UI with improved usability
- New bottom (or side) drawer UI for controls
- Imporved waterfall plot
- New custom Tuner-Wheel UI element
- Integrated in-app help system and extensive user manual
- Recordings are saved inside the App and can be downloaded/shared.
- Sampling and Demodulation while the app is in the background
- Support for CW (Morse) audio demodulation

version 1.13
 - Bookmark frequencies
 - Including hackrf_android 1.12 (support for rad1o)
 - Select unit for frequency and bandwidth (MHz,kHz,Hz) in jump dialog
 - Support for Marshmallow permission model
 - Many bugfixes

version 1.12

 - New setting to show frequencies on the horizontal axis relative to the center frequency

    - Bugfixes
